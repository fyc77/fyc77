# 广西旅游网站后台管理系统接口文档

## 1. 认证接口

### 1.1 登录
- **请求方式**：POST
- **请求路径**：/api/auth/login
- **请求参数**：
  - username: string (用户名)
  - password: string (密码)
- **响应**：
  ```json
  {
    "code": 200,
    "message": "登录成功",
    "data": {
      "token": "JWT令牌",
      "user": {
        "id": 1,
        "username": "admin",
        "role": "admin"
      }
    }
  }
  ```

### 1.2 登出
- **请求方式**：POST
- **请求路径**：/api/auth/logout
- **请求参数**：无
- **响应**：
  ```json
  {
    "code": 200,
    "message": "登出成功"
  }
  ```

## 2. 景点管理接口

### 2.1 获取景点列表
- **请求方式**：GET
- **请求路径**：/api/scenic-spots
- **请求参数**：
  - keyword: string (关键词搜索)
  - categoryId: number (分类ID)
  - status: string (状态：online/offline)
  - page: number (页码)
  - pageSize: number (每页条数)
- **响应**：
  ```json
  {
    "code": 200,
    "message": "获取成功",
    "data": [
      {
        "id": 1,
        "name": "桂林山水",
        "address": "广西桂林市",
        "level": "5A",
        "categoryId": 1,
        "ticketPrice": "200",
        "openingHours": "08:00-18:00",
        "description": "桂林山水甲天下",
        "traffic": "公交1路、2路",
        "contact": "0773-1234567",
        "status": "online",
        "createdAt": "2023-01-01T00:00:00Z"
      }
    ],
    "total": 10
  }
  ```

### 2.2 新增景点
- **请求方式**：POST
- **请求路径**：/api/scenic-spots
- **请求参数**：
  - name: string (景点名称)
  - address: string (地址)
  - level: string (等级)
  - categoryId: number (分类ID)
  - ticketPrice: string (门票价格)
  - openingHours: string (开放时间)
  - description: string (图文详情)
  - traffic: string (交通信息)
  - contact: string (联系方式)
- **响应**：
  ```json
  {
    "code": 200,
    "message": "新增成功",
    "data": {
      "id": 2,
      "name": "北海银滩",
      "status": "online"
    }
  }
  ```

### 2.3 编辑景点
- **请求方式**：PUT
- **请求路径**：/api/scenic-spots/:id
- **请求参数**：同新增景点
- **响应**：
  ```json
  {
    "code": 200,
    "message": "编辑成功"
  }
  ```

### 2.4 删除景点
- **请求方式**：DELETE
- **请求路径**：/api/scenic-spots/:id
- **响应**：
  ```json
  {
    "code": 200,
    "message": "删除成功"
  }
  ```

### 2.5 批量操作
- **请求方式**：POST
- **请求路径**：/api/scenic-spots/batch
- **请求参数**：
  - ids: array (景点ID数组)
  - action: string (操作类型：delete/online/offline)
- **响应**：
  ```json
  {
    "code": 200,
    "message": "操作成功"
  }
  ```

- **请求参数**：

  - page: number (页码)
  - pageSize: number (每页条数)

    "code": 200,
    "message": "获取成功",

      {
        "id": 1,
        "title": "桂林三日游攻略",
        "author": "游客123",

## 5. 数据统计接口

### 5.1 获取统计数据
- **请求方式**：GET
- **请求路径**：/api/statistics
- **请求参数**：
  - startDate: string (开始日期)
  - endDate: string (结束日期)
  - category: string (分类：all/scenic/guide/message)
- **响应**：
  ```json
  {
    "code": 200,
    "message": "获取成功",
    "data": {
      "totalScenic": 50,
      "totalGuide": 100,
      "totalMessage": 500,
      "totalVisit": 10000,
      "visitTrend": {
        "labels": ["1月", "2月", "3月"],
        "values": [1000, 1500, 2000]
      },
      "contentDistribution": [
        {"name": "景点", "value": 50},
        {"name": "攻略", "value": 100},
        {"name": "留言", "value": 500}
      ],
      "userActivity": {
        "labels": ["周一", "周二", "周三"],
        "values": [100, 120, 150]
      },
      "interactionRate": {
        "labels": ["景点", "攻略"],
        "likeValues": [200, 300],
        "commentValues": [150, 250]
      }
    }
  }
  ```

### 5.2 导出统计数据
- **请求方式**：GET
- **请求路径**：/api/statistics/export
- **请求参数**：同获取统计数据
- **响应**：Excel文件
