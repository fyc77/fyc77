# 广西旅游网站后台管理系统部署说明

## 1. 环境准备

### 1.1 系统要求
- Node.js 14.0+ 
- MySQL 5.7+
- NPM 6.0+ 或 Yarn

### 1.2 依赖安装

#### 后端依赖
```bash
# 进入后端目录
cd backend

# 安装依赖
npm install
# 或使用 yarn
yarn install
```

#### 前端依赖
```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install
# 或使用 yarn
yarn install
```

## 2. 数据库配置

### 2.1 创建数据库
```sql
CREATE DATABASE guangxi_tourism;
```

### 2.2 导入数据库结构
```bash
# 进入后端目录
cd backend

# 执行SQL文件
mysql -u root -p guangxi_tourism < config/database.sql
```

### 2.3 配置数据库连接
编辑 `backend/config/config.js` 文件，修改数据库连接信息：

```javascript
module.exports = {
  // 数据库配置
  database: {
    host: 'localhost',
    port: 3306,
    username: 'root',
    password: 'your_password',
    database: 'guangxi_tourism'
  },
  // JWT配置
  jwt: {
    secret: 'your_jwt_secret',
    expiresIn: '24h'
  },
  // 服务器配置
  server: {
    port: 3000
  }
};
```

## 3. 启动服务

### 3.1 启动后端服务
```bash
# 进入后端目录
cd backend

# 启动开发服务器
npm run dev
# 或使用 yarn
yarn dev

# 生产环境启动
npm start
# 或使用 yarn
yarn start
```

### 3.2 启动前端服务
```bash
# 进入前端目录
cd frontend

# 启动开发服务器
npm run dev
# 或使用 yarn
yarn dev

# 生产环境构建
npm run build
# 或使用 yarn
yarn build
```

## 4. 访问系统

### 4.1 开发环境
- 后端API地址：http://localhost:3000/api
- 前端管理系统：http://localhost:5173

### 4.2 生产环境
- 前端构建后，将 `frontend/dist` 目录部署到静态文件服务器
- 后端服务部署到Node.js服务器

## 6. 系统架构

### 6.1 技术栈
- **前端**：Vue3 + Element Plus + Axios + ECharts
- **后端**：Node.js + Express + Sequelize ORM
- **数据库**：MySQL

### 6.2 目录结构
```
guangxi-tourism-admin/
├── backend/           # 后端代码
│   ├── config/        # 配置文件
│   ├── controllers/   # 控制器
│   ├── middleware/    # 中间件
│   ├── models/        # 数据模型
│   ├── routes/        # 路由
│   ├── app.js         # 主应用文件
│   └── package.json   # 后端依赖
├── frontend/          # 前端代码
│   ├── public/        # 静态资源
│   ├── src/           # 源代码
│   │   ├── components/ # 组件
│   │   ├── views/      # 页面
│   │   ├── router/     # 路由
│   │   ├── main.js     # 入口文件
│   │   └── App.vue     # 根组件
│   └── package.json   # 前端依赖
└── docs/              # 文档
    ├── 需求说明.md     # 需求文档
    ├── 接口文档.md     # 接口文档
    └── 部署说明.md     # 部署文档
```

## 7. 安全配置

### 7.1 生产环境注意事项
- 修改 `config.js` 中的 JWT 密钥为强密钥
- 修改数据库密码为复杂密码
- 配置 HTTPS 证书
- 配置 CORS 策略

### 7.2 性能优化
- 启用数据库索引
- 配置缓存机制
- 优化图片上传和处理
- 启用 Gzip 压缩

## 8. 常见问题

### 8.1 数据库连接失败
- 检查数据库服务是否启动
- 检查数据库连接配置是否正确
- 检查数据库用户权限

### 8.2 前端访问后端API失败
- 检查后端服务是否启动
- 检查前端 API 基础路径配置
- 检查 CORS 配置

### 8.3 图片上传失败
- 检查 `uploads` 目录权限
- 检查文件大小限制配置
- 检查服务器磁盘空间
