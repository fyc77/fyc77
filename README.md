## 🧩 项目概述
**Introduction to Guangxi Tourism** 是围绕广西本土旅游特色打造的全栈开发项目，聚焦广西文旅数字化需求，完整覆盖从0到1的需求分析、设计开发、测试部署全流程。项目采用清新的绿色主题设计，打造集景点介绍、美食推荐、文化展示、旅游攻略于一体的一站式服务平台，助力广西文旅宣传。

##  核心项目详情
### Introduction to Guangxi Tourism（主项目）
> 项目定位：面向游客的广西特色旅游一站式服务平台
-  项目背景：针对广西丰富的山水、美食、文化旅游资源，打造集景点介绍、美食推荐、文化展示、行程攻略于一体的数字化平台，为游客提供全面的广西旅游指南，助力广西文旅宣传。
-  核心功能：
  - 首页轮播展示广西标志性景观（如桂林山水），快速吸引用户
  - 六大导航板块：首页、景点、美食、文化、攻略、关于，覆盖旅游全需求
  - 广西14个地市特色景点、美食、非遗文化内容展示与检索
  - 响应式设计，适配手机/电脑多端访问
  - 特色景点快捷入口，一键直达热门目的地
-  项目贡献：
  - 独立完成项目需求分析、UI设计与技术方案选型
  - 负责前后端全栈开发，实现核心业务逻辑与接口
  - 完成数据库设计、数据录入与系统测试优化
  - 部署上线并持续迭代功能

##  项目运行说明
### 1. 环境准备
- 安装 [Node.js](https://nodejs.org/)（推荐 v16+ 版本，自带 npm 包管理器）
- 安装 [MySQL](https://www.mysql.com/) 数据库（用于项目数据存储）
- 安装 Git（可选，用于克隆仓库）

### 2. 项目启动步骤
#### 前端页面启动
1.  克隆/下载本仓库代码到本地
    ```bash
    git clone https://github.com/FYC77/fyc77.git
    cd fyc77
    ```
2.  直接打开前端入口文件 `index.html`，即可在浏览器中访问项目页面
    > （若使用本地服务器，可通过 VS Code 的 Live Server 插件启动，访问 `http://127.0.0.1:5500`）

#### 后端服务启动
1.  进入后端项目目录
    ```bash
    cd server
    ```
2.  安装项目依赖
    ```bash
    npm install
    ```
3.  配置数据库连接（修改 `config/db.js` 中的数据库地址、账号、密码）
4.  启动后端服务
    ```bash
    node server.js
    ```
5.  服务启动成功后，后端接口将运行在 `http://localhost:3000`

### 3. 项目结构说明
├── index.html          # 项目首页
├── css/                # 样式文件目录
├── js/                 # 前端交互脚本目录
├── server/             # 后端服务目录
│   ├── server.js       # 后端入口文件
│   ├── config/         # 配置文件目录
│   └── routes/         # 接口路由目录
├── README.md           # 项目说明文档
└── .gitignore          # Git 忽略文件配置


### 4. 注意事项
- 首次启动需确保 MySQL 服务已正常运行，并完成数据库初始化
- 若修改后端端口，需同步更新前端接口请求地址
- 本地开发时，建议使用浏览器开发者工具（F12）排查前端控制台报错
