##  核心项目详情
### Introduction to Guangxi Tourism（主项目）
> 项目定位：面向游客的广西特色旅游一站式服务平台
-  项目背景：针对广西丰富的旅游资源，打造集景点介绍、美食推荐、文化展示、行程规划于一体的数字化平台，助力广西文旅宣传
-  核心功能：
  - 广西14个地市特色景点、美食、非遗文化内容展示与检索
  - 旅游攻略、行程规划工具，支持自定义路线
  - 响应式设计，适配手机/电脑多端访问
  - 数据可视化展示广西旅游热度、景点分布等信息
-  项目贡献：
  - 独立完成项目需求分析、原型设计与技术方案选型
  - 负责前后端全栈开发，实现核心业务逻辑与接口
  - 完成数据库设计、数据录入与系统测试优化
  - 部署上线并持续迭代功能
- 🔗 项目源码：本仓库即为项目完整源码
